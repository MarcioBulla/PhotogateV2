# Project Image Library

Curated image assets for the Photogate V2 documentation site and repository README.

## Folders

- `board/` - PCB photos and KiCad board renders.
- `schematics/` - Vector schematic exports from KiCad and the complete schematic PDF.
- `signals/` - Sensor and comparator signal diagrams.
- `mechanical/` - FreeCAD assembly screenshots.
- `experiments/` - Gate, pulse, pendulum, centripetal, cylinder, and spring timing diagrams.
- `firmware/` - ESP-IDF menuconfig screenshots.

## Source

These files were copied from `PhotogateV2_materials` and renamed with English, web-friendly filenames. Keep new public image assets in this folder tree so the website and README can use stable relative paths.
